# Andaime PRO - Locação de Andaimes de Elite (Rio de Janeiro)

🚀 **Projeto de Local SEO de alta performance desenvolvido com Astro 4.x e Tailwind CSS.**

Este site foi construído seguindo os mais altos padrões de engenharia de software e design de conversão, focado em dominar os resultados locais do Google para o setor de locação de equipamentos na região do Rio de Janeiro.

## 🛠️ Tecnologias Utilizadas

- **Astro 4.x**: Framework focado em performance e carregamento ultra-rápido.
- **Tailwind CSS**: Estilização moderna e responsiva de alta fidelidade.
- **Lucide Icons**: Conjunto de ícones premium.
- **Vercel Adapter**: Configurado para deploy escalável e seguro.
- **SEO Técnico**: Schema.org (LocalBusiness), Sitemaps, Meta Tags OG e Meta tags de busca local.

## ✨ Recursos Implementados

1.  **Nota 100 no PageSpeed**: Carregamento instantâneo para máxima conversão.
2.  **Design de Conversão**: Menu mobile indestrutível, botões de ação (CTA) vibrantes e fluxo direto para o WhatsApp.
3.  **SEO Local Blindado**: Configuração NAP (Name, Address, Phone) centralizada e Schema JSON-LD.
4.  **Componentização**: Estrutura modular, limpa e fácil de manter.
5.  **WhatsApp Premium**: Botão flutuante pulsante para atendimento imediato.

## 🚀 Como Executar Localmente

1. Clone o repositório
2. Instale as dependências: `npm install`
3. Execute em modo dev: `npm run dev`
